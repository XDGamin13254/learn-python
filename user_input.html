Python User Input

Handling user input is a crucial part of many programs. Python provides simple and effective ways to capture user input and use it in your code. Here's how you can work with user input in Python:

Using the input() Function

The input() function allows you to capture user input as a string:

name = input("Enter your name: ")
print(f"Hello, {name}!")

This code prompts the user to enter their name and then greets them.

Converting Input

Since the input() function returns a string, you might need to convert the input to other types:

age = input("Enter your age: ")
age = int(age)
print(f"You are {age} years old.")

This code captures the user's age and converts it to an integer before using it.

Handling Invalid Input

It's important to handle cases where the user provides invalid input. You can use a try-except block to manage this:

try:
    age = int(input("Enter your age: "))
    print(f"You are {age} years old.")
except ValueError:
    print("Please enter a valid number.")

Using input() in Loops

You can use the input() function within loops to repeatedly prompt the user for input until you get a valid response:

while True:
    age = input("Enter your age: ")
    try:
        age = int(age)
        break
    except ValueError:
        print("Please enter a valid number.")
print(f"You are {age} years old.")

This code will keep asking for the user's age until a valid number is provided.

Advanced Input Handling

For more advanced input handling, consider using regular expressions or third-party libraries to validate and parse user input.

import re

email = input("Enter your email: ")
if re.match(r"[^@]+@[^@]+\.[^@]+", email):
    print("Valid email address.")
else:
    print("Invalid email address.")

Arithmatic operators in python are very simple, although there are some key things to know before you try them out.
The first thing to know is the difference between one equals sign and two, (IE; = or ==) as the double equals sign checks if the numbers on both sides are equal (IE; 1 == 1 is true, while 1 == 2 is false.) while one equal sign is setting a variable equal to something, which is a topic that will be visited later.
The first operator other than the equal sign is the addition sign, it's the same as real life. (IE; 1 + 4 == 5) subtraction works the same (IE; 4 - 1 == 3) Mulitplication works the same, but uses the star sign instead of an x (3 * 6 == 18). Division uses the slash symbol (8/2 == 4).
This is the end of operators looking or acting the same as in real life, the first notable difference is exponents using to star signs instead of a carrot (2**2 == 4) Modulo uses the percent sign, and finds the remainder when dividing the first number by the second (5%3 == 2).

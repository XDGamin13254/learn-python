Working with Files in Python

In Python, working with files is a common task for many applications. Python provides several built-in functions and methods to handle files. Here's a comprehensive guide to working with files in Python:

Opening and Closing Files

Files can be opened using the open() function, which returns a file object. Files must be closed using the close() method to free up system resources.

# Open a file
file = open('example.txt', 'r')  # 'r' means read mode

# Perform file operations

# Close the file
file.close()

Reading from Files

There are several methods to read from a file, such as read(), readline(), and readlines().

# Read the entire file
file = open('example.txt', 'r')
content = file.read()
print(content)
file.close()

# Read file line by line
file = open('example.txt', 'r')
for line in file:
    print(line, end='')
file.close()

Writing to Files

Files can be written to using the write() or writelines() methods. Use the 'w' mode to write to a file, which will overwrite the file if it exists.

# Write to a file
file = open('example.txt', 'w')
file.write('Hello, world!')
file.close()

Appending to Files

To append content to an existing file, open the file in 'a' mode.

# Append to a file
file = open('example.txt', 'a')
file.write('\nAppended text.')
file.close()

Using the with Statement

The with statement is used for resource management and ensures that files are properly closed after their suite finishes, even if an exception is raised.

# Using the with statement
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

File Modes

When opening a file, you can specify the mode in which the file is opened. Common modes include:

  • 'r': Read (default mode).
  • 'w': Write (creates a new file or truncates an existing file).
  • 'a': Append (writes data to the end of the file).
  • 'b': Binary mode (e.g., 'rb' or 'wb' for reading or writing binary files).

Python Additions

Addition is one of the fundamental arithmetic operations, and it is straightforward in Python. Here's a quick guide on performing addition in Python:

Basic Addition

You can add numbers using the + operator. Here's an example:

a = 5
b = 3
result = a + b
print(f"The result of {a} + {b} is {result}")

This code will output: The result of 5 + 3 is 8.

Adding Multiple Numbers

You can add more than two numbers in a single expression:

result = 1 + 2 + 3 + 4
print(f"The result of adding 1, 2, 3, and 4 is {result}")

This will output: The result of adding 1, 2, 3, and 4 is 10.

Using Variables

You can also use variables to store numbers and add them:

num1 = 7
num2 = 10
total = num1 + num2
print(f"The total of {num1} and {num2} is {total}")

This will output: The total of 7 and 10 is 17.

Adding Floating-Point Numbers

Addition works the same way with floating-point numbers:

a = 1.5
b = 2.3
result = a + b
print(f"The result of {a} + {b} is {result}")

This will output: The result of 1.5 + 2.3 is 3.8.

Adding Numbers in a List

You can sum all the elements in a list using the sum() function:

numbers = [1, 2, 3, 4, 5]
total = sum(numbers)
print(f"The sum of the list is {total}")

This will output: The sum of the list is 15.

Handling User Input

You can also add numbers provided by the user. Here's an example that adds two numbers input by the user:

num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
total = num1 + num2
print(f"The total of {num1} and {num2} is {total}")

This code will prompt the user to enter two numbers and then output their sum.

Python Functions

Functions are a fundamental building block in Python programming. They allow you to encapsulate code into reusable blocks, making your programs more modular and easier to manage. Here's a quick overview of how to work with functions in Python:

Defining a Function

def my_function():
    print("Hello from a function")

This defines a function named my_function that prints a message.

Calling a Function

my_function()

This calls my_function and executes its code.

Function Parameters

Functions can take parameters, allowing you to pass values to them:

def greet(name):
    print(f"Hello, {name}!")
greet("Alice")  # Outputs: Hello, Alice!

Return Values

Functions can also return values using the return statement:

def add(a, b):
    return a + b
result = add(3, 5)
print(result)  # Outputs: 8

Default Parameters

You can define default values for parameters, making them optional:

def greet(name="Guest"):
    print(f"Hello, {name}!")
greet()       # Outputs: Hello, Guest!
greet("Bob")  # Outputs: Hello, Bob!

Keyword Arguments

Python allows you to use keyword arguments to make function calls more readable:

def describe_pet(animal_type, pet_name):
    print(f"I have a {animal_type} named {pet_name}.")
describe_pet(animal_type="hamster", pet_name="Harry")

Arbitrary Arguments

You can use *args and **kwargs to accept an arbitrary number of positional and keyword arguments:

def make_pizza(*toppings):
    print(f"Making a pizza with {toppings}")
make_pizza("pepperoni", "cheese", "olives")

Understanding Data Types in Python

Data types are a fundamental aspect of any programming language, and Python is no exception. Python supports various data types, each serving a specific purpose. Understanding these data types is essential for writing efficient and effective code.

Numeric Types

Python supports different types of numbers, including integers, floating-point numbers, and complex numbers.

  • int: Whole numbers, e.g., 5, -3, 42
  • float: Numbers with decimal points, e.g., 3.14, -0.001, 2.0
  • complex: Complex numbers with real and imaginary parts, e.g., 1+2j, 3-4j

Sequence Types

Sequences are collections of items, and Python supports several types of sequences:

  • list: Ordered, mutable collections of items, e.g., [1, 2, 3]
  • tuple: Ordered, immutable collections of items, e.g., (1, 2, 3)
  • range: Represents a sequence of numbers, e.g., range(5) (0 to 4)

String Type

Strings are sequences of characters used to represent text. In Python, strings are immutable.

text = "Hello, World!"
print(text)
print(text[0])
print(text[0:5])

Set Types

Sets are unordered collections of unique items. Python supports two types of sets:

  • set: Mutable sets, e.g., {1, 2, 3}
  • frozenset: Immutable sets, e.g., frozenset([1, 2, 3])

Mapping Types

Mappings are collections of key-value pairs. The primary mapping type in Python is the dictionary.

student = {"name": "Alice", "age": 20, "grade": "A"}
print(student["name"])
print(student.get("age"))

Boolean Type

The boolean type represents truth values, and it can be either True or False.

is_student = True
is_graduated = False

None Type

The None type is used to signify the absence of a value or a null value.

result = None

Type Conversion

Python supports explicit type conversion (also known as type casting). Here are some common conversions:

num = 10
str_num = str(num)  # Convert int to string
float_num = float(num)  # Convert int to float
int_num = int("20")  # Convert string to int
